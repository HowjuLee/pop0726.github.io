证明：根据归结原理
　　将待证明公式转化成待归结命题公式：
　　(P → Q) ∧～(～Q → ～P)
　　分别将公式前项化为合取范式：
　　P → Q ＝ ～P ∨ Q
　　结论求～后的后项化为合取范式：
　　～(～Q → ～P)＝ ～(Q∨～P) ＝ ～Q ∧ P
　　两项合并后化为合取范式：
　　（～P ∨ Q）∧～Q ∧ P
　　则子句集为：
　　{ ～P∨Q，～Q，P}
　　对子句集中的子句进行归结可得：
　　1. ～P∨Q
　　2. ～Q
　　3. P
　　4. Q，（1，3归结）
　　5. e，（2，4归结）
　　由上可得原公式成立。
