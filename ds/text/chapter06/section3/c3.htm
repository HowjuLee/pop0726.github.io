6.3.2 链式存储结构

　二、 三叉链表

二叉树的三叉链表存储表示
　　typedef struct TriTNode {
　　　ElemType data;
　　　struct BiTNode *Lchild, *Rchild; // 左、右孩子指针
　　　struct BiTNode *parent;　　　　　// 双亲指针
　　} *TriTree;

　　和二叉链表相同，表示整个二叉树只需要一个指向根结点的指针即可。和上页相同的二叉树的二叉链表如下图所示。

  

　三叉链表的结点结构：

parent
Lchild
data
Rchild

　　类似于线性表的双向链表，在二叉树的三叉链表中既有指示"后继"的信息，也有指示"前驱"的信息。
