Abort Interrupt
失效中断 失效中断属于异常中断，中断请求也可能发生在一条指令的执行过程中，但除非强制干预或系统重新复位，否则机器无法继续正常工作下去。
Access Locality
局部性原理 程序往往重复使用它刚刚使用过的数据和指令，分时间上的局部性和空间上的局部性。
Address Code
地址码 一条指令中给出操作数地址信息的字段称为地址码。
Address Mapping
地址映像 把用户用虚拟地址编写的程序按照某种规则装入到主存储器中，并建立多用户虚地址与主存实地址之间的对应关系，称为地址映像。
Address Translation 地址变换
在实际运行时，把多用户虚地址变换成主存实地址（内部地址变换）或磁盘存储器地址（外部地址变换）的过程称为地址变换。
Addressing Mode 寻址方式
寻找操作数及数据存储单元的方法称为寻址方式。
