Machine Data 机器数
把一个数连同其符号在内在机器中的表示加以数值化，这样的数称为机器数。
Machine Language 机器语言
计算机能识别并直接执行的语句，由二进制代码组成，称为机器语言。
"Massively Parallel Processing ,MPP" 大规模并行性并行处理
按照当前的标准，具有几百或几千台处理机的任何机器就是大规模并行处理系统。
Maximum degree of parallism 最大并行度
计算机系统在单位时间内能够处理的最大的二进制位数。
Memory 存储器
计算机的记忆部件。
Memory Addressing 主存寻址
主存寻址方式主要包括直接寻址方式、间接寻址方式和变址寻址方式等三种类型。
