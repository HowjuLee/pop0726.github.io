Scalability 可扩展性
系统规模增大或总负载大小变化时系统的适应能力。
Scalable Shared-memory Multiprocessing 可扩展共享存储多处理
S2MP是一种共享存储的体系结构，它支持简单的编程模型，系统使用方便，是对SMP系统在支持更高扩展能力方面的发展。
Scalar Processor 标量处理机
具有标量数据表示和标量指令系统的处理机称为标量处理机。
Segmentation Virtual Memory 段式虚拟存储器
一般按照程序的内容和函数关系把程序分成段，每段都有自己的名字，并且能够按照名称或序号来访问程序段。
Segmentation/Paging Virtual Memory 段页式虚拟存储器
对用户原来编写程序的虚拟存储空间采用分段的方法管理，而对主存储器的物理空间采用分页的方法管理。
Selector Channel 选择通道
选择通道可以认为是只有一个以成组方式工作的子通道，只有一套完整的硬件，它逐个为几个物理上连接的几台高速外围设备服务。
