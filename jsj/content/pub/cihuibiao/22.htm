Operating System 操作系统
操作系统是系统程序的集合，对系统的软硬件资源进行管理，为用户创造方便、有效、可靠的计算机工作环境。
Operation Code 操作码
一条指令中指示计算机所要执行的操作的字段称为操作码。
Optimal Replacement Algorithm 最优替换算法
选择将来最久不被访问的页面作为被替换的页面。
Out of Order Flowing 乱序流动
"在流水线的输入端，指令是按照它的地址从小到大顺序进入流水线的；但在流水线的输出端，指令执行完成的顺序可能是混乱的,这种控制方式称为乱序流动"。
Out－order Completion 乱序完成
指令的完成顺序不完全按照程序中的指令排列顺序进行，称为顺序完成。
Out-order Issue 乱序发射
指令的发射顺序不完全按照程序中的指令排列顺序进行的，称为乱序发射。
