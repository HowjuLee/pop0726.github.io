答案

　四种寻径方式为：
　（1）线路交换寻径：其传输时延用公式表示为
　　　T = (Lt/B)*D + L/B
　　　式中Lt为建立路径所需的小信息包长，
　　　L为信息包长，D为经过的结点数，
　　　B为带宽(以下同)。

　（2）存储转发寻径：传输时延公式为
　　　T = (L/B)*D + L/B = (D+1) * L/B

　（3）虚拟直通：传输时延公式
　　　T = (Lh/B)*D + L/B = (Lh*D + L)/B
　　　Lh是消息的寻径头部的长度，一般L>>Lh*D。
　　　所以公式可以近似为T = L/B

　（4）虫蚀寻径：传输时延公式
　　　T =Lf*D + L/B = (Lf/B)*D + L/B = (Lf*D+L)/B
　　　Lf是片的长度，Tf是片经过一个结点所需的时间，
　　　一般 L>>Lf*D,所以公式可以近似为 T = L/B
