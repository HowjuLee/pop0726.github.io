（3）集合类型（SET）
　　由0个或多个元素类型组成的无序集合。组成的元素必须是可区分的。

例：
　　　TypeSet ::= SET {
　　　　　　　　　a BOOLEAN,
　　　　　　　　　s INTEGER,
　　　　　　　　　n BIT STRING
　　　　　　　　　}
　　　valSet TypeSet :: = {
　　　　　　　　　a TRUE,
　　　　　　　　　N '1001'B,
　　　　　　　　　s 520
　　　　　　　　　}

（4）单一集合类型（SET OF）
　　由0个或多个同一类型元素组成的无序集合。

例：
　　　Words ::= SET OF VisibleString
　　　buildWords Words ::= {"entities","protocols","pdu"}
