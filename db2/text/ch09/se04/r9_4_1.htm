SQL Server 2000 数据转换服务 (DTS) 是一组图形工具和可编程对象，可以将取自完全不同源的数据析取、转换并合并到单个或多个目的。
　　DTS 包括一些创建、执行和管理的工具包：
　　・DTS 导入/导出向导，用于生成包以导入、导出和转换数据或复制数据库对象。
　　・DTS 设计器，一个图形化应用程序，可以构造包含复杂工作流、到异类数据源的多个连接和事件驱动逻辑的包。
　　在 SQL Server 企业管理器控制台树中的"数据转换服务"节点用于查看、创建、装载和执行 DTS 包、控制 DTS 设计器设置以及管理执行日志。包执行实用工具：
　　・dtswiz 实用工具通过使用命令提示选项启动 DTS 导入/导出向导。
　　・dtsrun 实用工具从命令提示运行包。
　　・DTS 运行实用工具 (dtsrunui) 可以使用对话框运行包。
　　・DTS 查询设计器，一个可视数据库工具，它使得在 DTS 设计器中生成查询更加容易。
　　数据转换服务DTS可以为如下数据源导入和导出数据:
　　・-OLEDB数据源，如Oracle, SQL Server等
　　・-ODBC数据源，如MVS上的DB2,Excel, Access等
　　・-ASCII文本文件；定长或有分隔符。
　　・-通过DTS Wizard的简单转换，或通过DTS Designer的复杂数据转换。
　　2. Microsoft 存储池(Reposity)
　　是共享元数据的存储区。存储的内容包括：
　　・-存储星型模式数据模型；
　　・-按源系统的数据元素关系分类；
　　・-记录数据的转换方式；
　　・-存储数据检索和复制的规则。
　　3 决策支持服务DSS
　　Microsoft DSS 把OLAP的处理功能和数据仓库的信息集成起来，可实现多维立方体的信息表示，进行片状或块状的信息分析。主要特点是：
　　・可访问任意与OLE DB 兼容的数据源；
　　・支持MOLAP(多维OLAP)、 ROLAP(关系OLAP)和 HOLAP(混合OLAP)；
　　・通过创建视图表建立SQL Server 和Excel之间的联系；
　　・支持移动的或离线的信息分析；
　　・其功能可以在桌面和整个企业中实现。
　　4 英语查询工具
　　提供用户用英语交互方便地访问数据仓库中的信息。
