3 执行存储过程
　　在SQL Server中，使用EXECUTE语句执行存储过程。EXECUTE语句格式如下：
　　EXEC[ute]
　　[@return_status =]
　　{ procedure_name[；number] | @procedure_name_var } [[@parameter =] {value | @variable[OUTPUT] | [DEFAULT]} ,…]
　　[WITH RECOMPILE] 　　在SQL Server中，执行存储过程的方法有两种：
　(1) 独立执行存储过程

　　例如，EXEC overdate_books
　(2) 在INSERT语句中执行存储过程


　例1，定义存储过程把从employee 表中查到的今天雇佣的雇员数据加载到customer表。存储过程的定义如下：
　　CREATE PROCEDURE employee_customer
　　AS
　　SELECT * FROM employee
　　WHERE hiredate = getdate()

　例2，在INSERT语句中调用存储过程employee_customer执行。
　　INSERT INTO customer
　　EXEC employee_customer
　
　4. 更改和删除存储过程
　(1) 更改存储过程：ALTER PROCEDURE
　　SQL Server中可以更改存储过程，就是用新定义的存储过程替换原来的定义。语句格式：
　　ALTER PROC[EDURE] <存储过程名> { 同定义}
　(2) 删除存储过程：DROP PROCEDURE
　　从当前数据库中删除用户定义的存储过程。语句格式：
　　DROP PROC[EDURE] <存储过程名>
