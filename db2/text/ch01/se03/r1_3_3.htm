1. 企业管理器：（Enterprise Manager）
　　Enterprise Manager可以在客户端和服务器端运行。它是图形化的用户界面，是服务器和客户端的主要管理工具。
　● 服务器端的SQL Enterprise Manager，可以管理服务器和数据库及数据库对象，设置对数据库对象的访问许可。其他功能如执行SQL Server的管理任务，创建数据库、备份和恢复数据库、数据转换、创建数据库图表和维护计划，执行各种Web管理和任务调度等。
　● 客户端的SQL Enterprise Manager，可用于 设计和创建数据库、查询数据等。

　　SQL Server提供了企业管理器作为数据库管理工具，绝大多数数据库管理工作可以在企业管理器中完成。企业管理器以树形结构管理SQL Server数据库服务器、数据库以及库中的对象。
　　执行步骤：打开"开始"菜单，从"程序"中选择 "Microsoft SQL Server"，再从程序组中选择"企业管理器"，如图所示。
