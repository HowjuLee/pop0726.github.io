2．SQL SERVER 的安装需求：
　　下表说明安装 Microsoft SQL Server 2000 或 SQL Server 客户端管理工具的硬件要求。
　(1) 硬件要求
　● 计算机 Intel或兼容机 ，Pentium 166 MHz 或更高。
　● 内存 (RAM)
　　企业版：至少 64 MB，建议 128 MB 或更多
　　标准版：至少 64 MB
　　个人版：Windows 2000 上至少 64 MB，其它所有操作系统上至少 32 MB
　　开发版：至少 64 MB
　● 硬盘空间
　　SQL Server 数据库组件：95 到 270 MB，一般为 250 MB
　　Analysis Services：至少 50 MB，一般为 130 MB
　　English Query：80 MB
　　仅 Desktop Engine：44 MB
　● 监视器 VGA 或更高分辨率：
　　SQL Server 图形工具要求 800x600 或更高分辨率。
　● 定位设备： Microsoft 鼠标或兼容设备 。
　● CD-ROM 驱动器等。

　　在实际教学过程中操作系统推荐使用 WINDOWS 2000 SERVER 或 ADVANCE SERVER,
　　SQL SERVER 使用企业版或标准版。
　　原因：标准版和企业版支持大型数据库系统，而个人版用于临时存储和移动计算。在安装企业版和标准版的时候，需要的操作系统为WINDOWS 2000 SERVER 或 ADVANCE SERVER等，如上表所示。
　　在实际安装时，应尽量考虑到硬盘空间的使用问题，因为使用数据库系统的目的在于存储和管理数据，用户应为将来存储的数据留出足够的空间。
　　在安装SQL SERVER2000时，还需要 WINDOWS NT SP5 和IE5.0,随SQLSERVER 2000的光盘提供。只需要在安装时选择最小安装就可满足SQL SERVER2000使用需求。
　　安装前的注意事项：
　● 关闭所有可能与SQL SERVER2000有关的服务和应用程序。包括连接到ODBC的应用程序，以及INTERNET INFORMATION SERVER(IIS)
　● 关闭WINDOWS的事件查看器。
　● 关闭任何管理或修改注册表的应用程序。例如Regedit。
　● 在安装之前，首先用拥有系统管理员权限的帐号登陆操作系统。
