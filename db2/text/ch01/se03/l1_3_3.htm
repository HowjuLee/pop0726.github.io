1.3.3 SQL Server的管理工具：
　SQL Server的管理软件可以在客户端和服务器端同时运行，主要包括：
　① Microsoft Management Console的SQL Server Enterprise Manager的附加程序
　② SQL Server Query Analyzer
　③ SQL Server命令方式管理工具
　④ SQL Server帮助和SQL Server在线手册
　⑤ SQL Server Service Manager：启动、停止和暂停SQL Server服务的图形实用程序。
　1. 企业管理器（Enterprise Manager）
　2. 查询分析器（Query Analyzer）
　3. SQL Server的其他管理工具
　4. SQL Server命令方式管理工具
　5. SQL Server帮助和SQL Server联机丛书
　6. SQL Server附带产品功能
