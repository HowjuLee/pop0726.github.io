1. SQL Server2000的版本：
　・ SQL Server2000企业版：
　　该版本支持了SQL Server的全部特性，为关键商业任务和Internet方案提供高级可伸缩性和可靠的功能，可作为大型web站点，企业OLTP(联机事务处理)以及数据仓库系统的数据库服务器。
　　由于它可以充分利用最高端硬件，支持32颗CPU和64GB内存，在执行多处理（Multiprocessing）时可以使系统性能得到显著的提高。
　　多处理器是指服务器拥有多个CPU的情况，WINDOWS NT使用对称多处理，可以将操作系统和应用程序的进程分散在系统所有的CPU中运行。

　・ SQL Server2000标准版：
　　可用于小型或中型组织的经济性版本，除去了企业版中的高级可伸缩性和一些高级的分析功能，标准版最多支持4颗CPU和2GB内存。
　　

　・ SQL SERVER2000个人版：
　　它包括了全套的管理工具和标准版的大多数功能，并为个人使用进行了优化，除了在服务器操作系统上运行，也可以在非服务器操作系统上运行。该版本支持双处理器系统，性能为单个用户及小型工作组进行了优化。
　　该版本支持在非服务器的操作系统上运行，包括有WINDOWS 2000 PROFESSIONAL,WINDOWS NT WORKSTATION4.0 以及 WINDOWS 9X。当并发用户超过五个时，性能会随工作负荷的增加而降低。

　・ SQL SERVER2000开发人员版：
　　该版本允许开发人员在SQL SERVER 的顶端生成任意类型的应用程序。包括企业版的所有功能。
