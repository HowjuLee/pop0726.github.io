7.7 数据库服务程序

7.7.1 SQL Server代理服务

　　关数据库管理工作是一项重要而繁琐的任务，为了减轻管理员负担，SQL Server提供了SQL Server Agent服务。SQL Server Agent能够自动执行管理员预先安排的管理任务、监视SQL Server事件、并根据事件触发警报。
��SQL Server Agent在Windows NT下是作为一个服务运行的。SQL Server Agent和以前版本的SQL Server Executive相比，在功能上大大增强了。利用SQL Server Agent可以降低系统管理开销。
��SQL Server 代理服务能够自动执行管理员安排的每天必须进行的固定不变的任务，还可以在服务器发生异常事件的时候，自动发出通知，以便让操作人员及时获得信息，并作出处理。

SQL Server代理服务由三个部分组成：
��● 作业(job)：需要执行的管理工作。
��● 操作员(operator)：接收警报事件通知的操作人员。
��● 警报(alert)：警报是管理员所定义的对SQL 事件的响应动作。
