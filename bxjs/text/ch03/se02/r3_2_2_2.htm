　　在很多的算法中，输出集中的数据之间的存在复杂的依赖关系，这种情况下，对输出集按照元素进行划分不再可行。对有的算法来说，它们的计算可以被重新组织为下面的多级计算：
　　

　　其中每级的输出数据之间没有依赖关系（它们对应的计算可以并行进行）。此时可以对中间数据（每级的输出）进行数据划分。
比如下面的计算前序和的算法：
　　　　　PrefixSum(A, n, S)
　　　　　{
　　　　　　for (j=0; j<n; j++)
　　　　　　　S[j] = A[j];

　　　　　　for (i=1; i<n; i*=2) {
　　　　　　　for (j=n-1; j>=i; j--)
　　　　　　　　S[j] += S[j-i];
　　　　　　}
　　　　　}
